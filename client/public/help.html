How to use Aleph

To use Aleph, go to the home page by clicking "Home" on the navigation bar above.

Once you're on the home page, look for the prompt text input area at the bottom of the screen.

Type in your prompt. It can be something like "Give me a cheesecake recipe" or "Write code for a HTTP GET request in TypeScript".

Press enter or click "Send" and wait a few moments. Aleph should be able to answer your prompt and display it on the screen.
